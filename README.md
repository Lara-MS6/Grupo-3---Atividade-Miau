# Sistema de Cadastro Geográfico

Sistema minimalista CLI desenvolvido em TypeScript com persistência em banco de dados SQLite e ORM Drizzle que permite o gerenciamento de UFs, Cidades e Regiões.

## Tecnologias

- **TypeScript** - Linguagem principal
- **Node.js** - Ambiente de execução
- **Drizzle ORM** - Mapeamento objeto-relacional
- **SQLite** - Banco de dados local

## Pré-requisitos

- Node.js (versão 18 ou superior)
- npm ou yarn
